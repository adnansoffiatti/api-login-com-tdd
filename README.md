# Projeto

API de Login, utilizando TDD

## Para clonar repositório:

git clone https://github.com/adnansoffiatti/api-login-com-tdd.git

## Dependências usadas:

-   Express
-   Mongoose
-   Dotenv
-   Bcrypt
-   JSONWebtoken
-   Jest
-   Supertest

Types utilizado para autocomplete:

-   @types/jest

## Para baixar depedências:

npm install
